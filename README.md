# 다형성을 활용한 회원주문 프로젝트
회원가입을 통한 로그인 및 상품 관리를 하는 게시판 프로젝트 입니다.

다형성과 의존성 주입을 통해 서버와 DB를 각각 연동을 해봤습니다.
* 순수 jdbc 구현
* Mybatis 연동
* 순수 JPA 구현
* SpringData JAP = > Querydsl 

## 개발환경
* Intellij IDEA Ultimate 
* Java 11
* Spring Boot 2.6.5

## 기술 스택
Spring Boot


* Spring Data JPA
* Querydsl
* Thymeleaf
* Spring Web
* Lombok
* Validation
* H2 Database
* MySQL Driver
* JdbcTemplate
